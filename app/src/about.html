Software

Code and data for RUPEE are available at https://github.com/rayoub/rupee

This website and the RUPEE backend make use of the following freely available software.

BioJava
Parsing and aligning protein structures
jQuery
JavaScript library
Bootstrap
HTML styling
PostgreSQL
Backend database
ChemDoodle
3d alignments
